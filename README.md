# Fintech Behavioral Analytics Platform

A portfolio project demonstrating end-to-end analytics engineering:
Plaid Sandbox API → GCS → BigQuery → dbt → Looker Studio

## Stack
- **Extraction:** Python (Plaid Sandbox API)
- **Raw Storage:** Google Cloud Storage
- **Warehouse:** BigQuery
- **Transformation:** dbt (medallion architecture)
- **Orchestration:** Apache Airflow
- **Dashboards:** Looker Studio
